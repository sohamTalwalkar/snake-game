# SnakeGame
This is a snake game, where the snake eats food to grow its size and this increases the player's score. The game runs on desktop browsers and is controlled using the keyboard arrow keys.

# About the Game
- The game uses a 25Ã—25 grid created dynamically using div elements
- The snake moves at fixed time intervals
- Each part of the snake is stored as a node in a linked list
- The snake grows when it eats fruit
- The game ends when the snake collides with itself

# Tech Stack
- HTML
- CSS
- JavaScript

# How to run
- Clone the repository
- Open SnakeGame.html in a browser
- Click start and use arrow keys to move the snake
